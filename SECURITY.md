# Security Policy

## Supported Versions

We only provide security updates for the latest stable release of MIDITypist.

| Version | Supported          |
| ------- | ------------------ |
| v1.0    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in MIDITypist, please DO NOT open a public issue. Instead, please report it privately.

You can report vulnerabilities by:
1.  Opening a **Private Vulnerability Report** on GitHub (if enabled for the repo).
2.  Contacting the maintainer directly at [INSERT EMAIL OR LINK].

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary. Please provide a proof-of-concept or clear steps to reproduce the issue.

Thank you for helping keep MIDITypist secure!
